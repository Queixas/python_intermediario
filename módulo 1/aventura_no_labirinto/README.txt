# Aventura no Labirinto

Aventura no Labirinto é um jogo interativo em Python onde os jogadores exploram um labirinto em busca de um item escondido. O jogo oferece uma experiência divertida e desafiadora, permitindo que os jogadores se movimentem em um ambiente gerado aleatoriamente.

## Visão Geral

Neste jogo, você deve navegar por um labirinto, coletar itens e encontrar o objetivo antes de ficar sem movimentos. Com uma interface de linha de comando (CLI) simples, o jogo é fácil de usar e acessível a todos.

## Recursos

- Labirintos gerados aleatoriamente.
- Movimentação do jogador através das teclas de direção.
- Animações de vitória com fogos de artifício.
- Opções de personalização através da linha de comando.
- Música ambiente com opção para ativar/desativar.
- Níveis de dificuldade ajustáveis.
